# agent-dev-utils

Claude Code plugin marketplace for AI agent development.

## Quick Start

```bash
# Install the marketplace
/install-plugin harrytruong0804/agent-dev-utils

# Install a specific plugin
/install-plugin harrytruong0804/agent-dev-utils@openai-agents-tracing
/install-plugin harrytruong0804/agent-dev-utils@context-engineering
```

## Plugins

| Plugin | Description |
|--------|-------------|
| `openai-agents-tracing` | Self-hosted tracing dashboard for OpenAI Agent SDK |
| `context-engineering` | Context engineering audit for AI agent codebases |
